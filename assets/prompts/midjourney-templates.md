# MidJourney Prompt Templates

## Style Guidelines

All prompts should follow these rules:
- NO text, letters, or numbers in images
- Professional consulting aesthetic
- Clean, minimalist design
- Corporate color palette (adjustable per theme)
- Abstract representations, not literal scenes
- 16:9 aspect ratio

## Prompt Structure

```
[Subject/Concept], consulting presentation style, clean minimalist design, 
abstract geometric visualization. [Color palette]. Professional corporate 
aesthetic. No text, no letters, no numbers, no labels. [Additional details]. 
--ar 16:9 --style raw --v 6
```

## Templates by Slide Type

### Revenue/Growth
```
Abstract upward trending visualization, geometric shapes increasing in size 
from left to right, bar chart abstraction. Navy blue primary with light blue 
accents. Clean minimal design. White background with subtle grid pattern. No text or numbers.
```

### Market/Mapping
```
Abstract map visualization, geometric regions, strategic positioning concept. 
Navy blue primary color with accent highlights. Minimalist corporate style. 
Connected nodes or network pattern.
```

### Process/Flow
```
Abstract process flow diagram, geometric arrows and pathways, sequential 
steps visualization. Navy blue and light blue color scheme. Clean lines, 
minimal design. Directional flow from left to right. No text.
```

### Comparison
```
Split composition comparison, two distinct geometric forms side by side, 
before and after concept. Navy blue primary, coral accent for contrast. 
Balanced minimalist design.
```

### Timeline
```
Horizontal timeline abstraction, geometric milestones, sequential progression. 
Navy blue elements on white background. Clean minimal style with connected nodes.
```

### Challenges/Problems
```
Abstract obstacle or barrier visualization, geometric challenge representation. 
Navy blue with coral/red accent for warning. Professional corporate aesthetic.
```

### Solutions/Success
```
Abstract solution visualization, checkmark or success symbolism, geometric 
completion concept. Navy blue primary with green or gold accent. Positive, 
upward energy.
```

### Data/Analytics
```
Abstract data visualization, geometric chart elements, analytical concept. 
Grid pattern background, navy blue data points. Clean technical aesthetic.
```

### Team/People
```
Abstract human figures or teamwork concept, geometric silhouettes, collaboration 
visualization. Navy blue with warm accent color. Professional, inclusive aesthetic.
```

### Strategy/Planning
```
Abstract chess piece or strategic board visualization, geometric planning 
concept. Navy blue primary with gold or amber accent. Thoughtful, analytical mood.
```
