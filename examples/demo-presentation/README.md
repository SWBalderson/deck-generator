# Demo Presentation

Example source documents for testing the deck-generator skill.

## Files

- `q4_strategy_brief.md` - Strategic review document with narrative
- `market_data.csv` - Quarterly performance data
- `competitor_analysis.pdf` - Competitor comparison (placeholder)
- `why-schools-need-decks-example.pdf` - Exported sample deck generated from `source_docs/article.md`

## Usage

```bash
# Run skill and select these files
# Expected output: 6-8 slide presentation covering Q4 performance
```

## Expected Slides

1. Title slide
2. Executive Summary (Q4 exceeded targets by 15%)
3. Financial Performance (waterfall or bar chart)
4. Market Expansion (map or growth chart)
5. Operational Metrics (efficiency gains)
6. Challenges & Mitigation (2-column layout)
7. Q1 2025 Priorities (timeline/Gantt)
8. Conclusion/Next Steps
